# MYMQOS - 操作系统课程设计

基于 **LoongArch64** 架构开发的操作系统内核，作为操作系统课程设计项目。


## 目录结构

```text
kernel/
├── drv/        # 设备驱动 (显卡, 字体)
├── excp/       # 异常与中断处理 (汇编入口, C处理函数)
├── include/    # 头文件 (寄存器定义, 函数声明)
├── init/       # 内核启动代码 (head.S, main.c)
├── mm/         # 内存管理 (物理页分配)
└── Makefile    # 编译脚本
```
